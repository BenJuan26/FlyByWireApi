## FlyByWire Simulations API

## Developing

Please make sure you have:

NodeJS 14 - [Homepage](https://nodejs.org/en/)

```bash
# install all dependencies
$ npm install

# start MySQL and Redis using docker-compose
$ npm run compose

# watch mode
$ npm run start:dev
```
