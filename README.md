## FlyByWire Simulations API

### Developing

Run the `setup.sh` script to create the required secrets for the service to connect to a database.

```bash
# install all dependencies
$ npm install

# start mysql using compose
$ npm run compose

# watch mode
$ npm run start:dev
```
